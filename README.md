# Running-from-Philosophy

For around 95% of Wikipedia articles, successively following the first link on the page will eventually lead you to the article for Philosophy.   This is a game about running from the underlying, inescapable nature of abstraction and Platonic reductionism.

You have sixty seconds to come up with as many topics as possible.  Each of these topics will be awarded a score equal to the number of times we can click on the first link of successive articles before we arrive at Philosophy.  Your overall score will be the sum of these topic scores.
